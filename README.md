# BookAnalysis

## Overview

As a developer working toward a career in machine learning engineering, I built this program to explore and analyze a large GoodReads dataset using Python and Pandas. The analysis investigates two questions: whether book length correlates with average reader ratings, and which popular books receive the most polarized reviews. This project is the first phase of The Shelf — an intelligent book recommendation engine I am building throughout 2026.
